# bitsensing

Radar products and developer resources from bitsensing.

This GitHub organization is the home of public SDKs, ROS 2 drivers, documentation, examples, and release notes for bitsensing radar products.

## Featured Developer Resources

- [`afi920-sdk`](https://github.com/bitsensing/afi920-sdk) - C/C++ and Python SDK for AFI920
- [`afi920-ros2-driver`](https://github.com/bitsensing/afi920-ros2-driver) - ROS 2 driver for AFI920
- [`bitsensing.github.io`](https://github.com/bitsensing/bitsensing.github.io) - Developer documentation and integration guides

## Product Portfolio

### Automotive

- `AIR4D`, model `AFI920` - 4D imaging radar for advanced perception and autonomy
- `ADAS Kit` (`AXM720`) - radar-based safety solution for commercial vehicles

### Smart Cities

- `ITM221` - radar and camera fusion edge device for intelligent traffic infrastructure

### Healthcare

- `MOD611` / `MOD620` - contactless radar sensing solutions
- `SQM` - sleep quality management API service

# Support

## GitHub Issues

Use GitHub Issues for:

- reproducible bugs
- documentation corrections
- feature suggestions

Do not use GitHub Issues for:

- confidential customer information
- security vulnerabilities
- commercial inquiries
- product evaluation or access requests

## Product and Commercial Support

For product, commercial, and integration inquiries:

- tech-support@bitsensing.com

## Security Reports

Do not report security issues in a public issue tracker.

- security@bitsensing.com

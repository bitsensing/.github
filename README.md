# bitsensing .github

Organization-wide profile and default community health files for the `bitsensing` GitHub organization.

## Contents

- `profile/README.md`: organization profile shown on the GitHub organization page
- `CONTRIBUTING.md`: contribution guidelines
- `SECURITY.md`: security reporting policy
- `SUPPORT.md`: support policy
- `.github/ISSUE_TEMPLATE/`: default issue templates
- `.github/pull_request_template.md`: default pull request template
